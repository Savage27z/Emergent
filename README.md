# EMERGENT

A browser-based sandbox world where every NPC is an autonomous AI agent — memory, reflection, plans, opinions. No scripts.

90-day build-in-public challenge · Day 1: July 6, 2026 → Launch: October 3, 2026.

## Run it

```
npm install
npm start
```

Open http://localhost:3000 — WASD to move.

## Stack

Three.js · Node.js + Socket.io · SQLite · LLM-driven agents (generative-agents pattern)
